# 3.1.概述

本章介绍了 FreeBSD 操作系统的基本命令和功能。这些章节中的大部分内容对于任何类 UNIX® 操作系统都是适用的。建议 FreeBSD 新用户仔细阅读本章。

阅读本章后，你将会了解：

* 如何使用和配置虚拟控制台。
* 如何在 FreeBSD 上创建和管理用户和组。
* UNIX® 文件权限和 FreeBSD 文件标志的工作原理。
* 默认的 FreeBSD 文件系统布局。
* FreeBSD 磁盘结构。
* 如何挂载文件系统和卸载文件系统。
* 进程、守护程序和信号是什么。
* Shell 是什么，如何修改默认登录环境。
* 如何使用基本文本编辑器。
* 什么是设备和设备节点。
* 如何阅读手册页面以获取更多信息。
